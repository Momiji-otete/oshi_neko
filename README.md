# <推し猫>

## サイト概要
### サイトテーマ
自分が飼っている猫の写真を投稿したり、みんなの猫を見て好きな猫を見つけるSNSサイト

### テーマを選んだ理由
猫が好きで、飼っている猫を見てほしいときもあるし、ほかの人が飼っているかわいい猫を見て癒されたいと思うときがあるから

### ターゲットユーザ
猫が好きな人

### 主な利用シーン
猫を見て癒されたいとき、飼っている猫を見せたくなったとき
